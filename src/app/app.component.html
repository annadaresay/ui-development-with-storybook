Demo: UI development with Storybook

By Anna Viklund 2022-01-18

What is it?

  • From webpage: "Storybook is an open source tool for building UI components and pages in isolation. It streamlines UI development, testing, and documentation."
  • Design, build and organize UI components
  • Open source
  • Integrates with most popular JavaScript UI frameworks
  • Runs alongside the application in development

Why use it?

  • Helps you build UI components isolated from the rest of your app
  • Helps you document components for reuse
  • Helps you test components to prevent bugs (manually and automatically)
